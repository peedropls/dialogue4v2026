# 🎮 INÍCIO RÁPIDO - AUDIO PLAYER

## 🚀 Comece em 30 SEGUNDOS

### Opção 1: Via Código (Mais Rápido!)
```csharp
// Abra o Console e execute:
AudioManager.Instance.PlayMusicByIndex(0);
```

### Opção 2: Via Editor
1. Selecione GameOject com `AudioPlayer`
2. No Inspector → "Teste em Play Mode"
3. Digite um número e clique botão
4. Pronto! A música toca em tempo real durante o jogo!

### Opção 3: Via UI (Mais Completo)
1. Crie GameObject vazio "AudioUISetup"
2. Adicione componente `AudioUIAutoSetup`
3. Menu de contexto (direita) → "Setup Automático da UI de Áudio"
4. Configure `AudioUIController` no Inspector
5. Execute o jogo e controle a música com botões!

## ⚡ ATALHOS DE TECLADO

| Tecla | Ação |
|-------|------|
| `0-9` | Toca música no índice 0-9 |
| `E` | Play |
| `P` | Pause |
| `R` | Resume |
| `S` | Stop |
| `←` | Anterior |
| `→` | Próxima |
